Cloud ID: 4087a625-67ca-4bbe-b2f0-3d406fdfed61 · Project AB
Browsers can't call Jira directly (CORS). A Cloudflare Worker acts as a secure proxy — free, no server needed.
☁️ Cloudflare Worker setup (free, ~5 min):
1. Go to workers.cloudflare.com → sign up free → Create Worker
2. Delete default code → paste the jira-proxy-worker.js file (download below)
3. Click Settings → Variables → Add variable:
   → JIRA_EMAIL = your Jira email
   → JIRA_TOKEN = your Jira API token (get token ↗)
4. Edit line 13 in the worker: replace YOUR-USERNAME.github.io with your actual GitHub Pages URL
5. Click Deploy → copy the worker URL (e.g. https://jira-proxy.YOUR-NAME.workers.dev)
6. Paste it below → Save
